my journey

a quantum physicist turned programming hobbyist (kind of).

  1. 2023 the start

    i was at UBC hoping to be a quantum physicist but i felt like the problems i was solving didn't have any impact in the real world. so i left and started learning C on my own. shortly after, i joined BCIT's Computer Systems Technology program in hopes of getting hands-on experience with building applications and solutions within teams.

  2. 2024 getting reps in

    the fast paced learning at BCIT forced me to go deep into programming fundamentals, object-oriented programming, networking, operating systems, web development, and much more. it was really hard but i enjoyed knowing more and more about the software running our world, and all the different ways to create it.

  3. 2026 shipping for real

    now i was learning industry standard frameworks, tools, and tech stacks. a lot of my earlier projects were local or hosted with some 3rd party service, but now i had to learn how to deploy it myself using either cloud services or in-house infrastructure. real production environments was a real learning curve for me, but having the responsibility of managing uptimes and deployments made me more confident in my abilities.
